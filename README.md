# Dungeon Crawler

A new dungeon crawler game project.

## Status
- **Phase**: Initialization / Prototype
- **Tech Stack**: Python (Flask) + HTML5/JS

## How to Run (Local)
1. Install dependencies: `pip install -r requirements.txt`
2. Run app: `python app.py`
3. Visit: `http://localhost:5000`

## How to Run (Docker)
1. `docker-compose up --build`
2. Visit: `http://localhost:5000`


## Features
- Retro "Terminal" UI
- Persistent Game World (`state.json`)
- Grid-based Movement
- Dungeon Master System (Skeleton)
